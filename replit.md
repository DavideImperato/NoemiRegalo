# 🎉 Pagina di Compleanno per Noemi - 30 Anni

## Panoramica
Una pagina web interattiva e animata creata per celebrare il 30° compleanno di Noemi. La pagina presenta un design elegante e festoso con animazioni dolci e un'esperienza interattiva sorprendente.

## Caratteristiche
- Sfondo con gradiente pastello animato che cambia dolcemente
- Effetti confetti decorativi che cadono continuamente
- Titolo "🎉 Buon 30° Compleanno Noemi! 🎂" con tipografia elegante
- Pacco regalo rosa con animazione di oscillazione (swing)
- Interazione al clic: il pacco scompare e rivela un messaggio sorpresa e un video
- Design completamente responsive (funziona su desktop, tablet e mobile)
- Tutto realizzato in HTML, CSS e JavaScript puro (nessun framework)

## Come Utilizzare il Video
Per aggiungere il tuo video personalizzato:

1. Carica il tuo file video nella directory principale del progetto
2. Rinomina il file come `video.mp4` oppure
3. Modifica il file `index.html` alla riga dove c'è `<source src="video.mp4"` e cambia il nome del file

Il video apparirà automaticamente quando l'utente clicca sul pacco regalo.

## Struttura del Progetto
- `index.html` - File principale con tutto il codice (HTML, CSS, JavaScript)
- `video.mp4` - Il tuo video personalizzato (da caricare)

## Tecnologie
- HTML5
- CSS3 (animazioni, gradients, transizioni)
- JavaScript vanilla
- HTML5 Video
